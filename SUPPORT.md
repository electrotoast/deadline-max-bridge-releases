# Support

Use this repository for release downloads and end-user feedback.

## Where To Go

- Installation failures: open an **Install Feedback** issue.
- Reproducible bugs after install: open a **Bug Report** issue.
- General setup questions: start a GitHub Discussion.

Latest downloads:

https://github.com/electrotoast/deadline-max-bridge-releases/releases/latest

## What To Include

For install issues, include:

- release version, such as `v0.1.1`
- whether you used the setup EXE or manual zip
- Windows version
- Deadline version
- 3ds Max version
- whether the current-user submitter was installed
- relevant output from `%TEMP%\DeadlineMaxBridge-Install-*.log`

For Worker or submitter issues, include:

- Deadline Worker log excerpt
- submitter path used, such as `Deadline > Submit Max To Deadline`
- renderer and plugin versions involved, if relevant
- whether the issue happens on one machine or multiple Workers

Do not post credentials, license keys, internal hostnames, or private project
paths unless you have intentionally sanitized them.
